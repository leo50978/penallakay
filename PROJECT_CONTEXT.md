# Contexte du Projet

## Concept du jeu

Le projet est un jeu de penalty 2D simple, realise en HTML, CSS et JavaScript.
Le but est de proposer une experience claire, immediate et amusante, facile a comprendre des la premiere interaction.

## Boucle principale

Le fonctionnement de base repose sur une boucle tres simple :

1. Le joueur choisit une direction de tir.
2. Le joueur tire.
3. Le gardien plonge dans une direction.
4. Le jeu determine s'il y a but ou arret.
5. Le score est mis a jour apres plusieurs tirs.

## Intention produit

Le projet doit rester simple au debut pour permettre un developpement rapide et propre.
Cela veut dire :

- peu de mecaniques au lancement ;
- une structure facile a maintenir ;
- des choix techniques simples ;
- des bases solides pour ajouter des evolutions plus tard.

## Evolutions possibles plus tard

Une fois la premiere version jouable terminee, le projet pourra evoluer avec :

- de meilleurs assets visuels ;
- des animations plus fluides ;
- plus de retours visuels et sonores ;
- un mode ou le joueur devient gardien ;
- des series de tirs plus longues ou avec difficulte progressive.

# Taches du Projet

## Roadmap initiale

### Etape 1

Creer une structure HTML, CSS et JavaScript propre.

### Etape 2

Mettre en place le terrain, le but, le ballon, le joueur et le gardien avec des placeholders.

### Etape 3

Creer le systeme de tir avec le choix d'une direction.

### Etape 4

Ajouter le mouvement du gardien selon une direction choisie.

### Etape 5

Ajouter le score et le suivi de plusieurs tirs.

### Etape 6

Integrer GSAP pour rendre les animations plus fluides.

### Etape 7

Remplacer les placeholders par de vrais assets visuels.

### Etape 8

Faire le polish final :

- ajustements visuels ;
- fluidite ;
- clarte de l'interface ;
- ressenti global du gameplay.

## Priorite

L'ordre de travail doit rester progressif.
On valide d'abord une base jouable, puis on ameliore la presentation et les animations.
